# Changelog

All notable changes to the Hain desktop app are documented here. This log starts
with the first public desktop release. The terminal tools (CLI and TUI) ship
alongside each release as additional assets.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and Hain aims to follow [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

The first public desktop release.

### Added

- Create a private space and sync a folder directly between your devices and the
  people you share with, with no cloud in between.
- Join a space by pasting a share key, with sync progress and peer status
  visible as it happens.
- Signed and notarized macOS builds for Apple Silicon and Intel, delivered as
  DMG.
- Native Linux builds for x64 and arm64, delivered as AppImage and deb.
- Automatic updates: background updates on macOS, and update notices on Linux.
- A `.sha256` checksum published next to every asset.

[Unreleased]: https://github.com/hainlabs/hain-releases/releases
